# CPP-Template

https://sourceforge.net/projects/mingw/

### Make an EXE

```bash
g++ main.cpp -o main.exe
```

or Run: exemake.bat

### Make a DLL

```bash
g++ main.cpp -o main.dll
```

or Run: dllmake.bat

Run: runmain.bat
